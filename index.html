Projeto Agrinho

O Projeto Agrinho é uma iniciativa promovida pelo Sistema FAEP/SENAR-PR (Federação da Agricultura do Estado do Paraná e Serviço Nacional de Aprendizagem Rural) voltada para a educação e o desenvolvimento social e ambiental nas áreas rurais do estado do Paraná. O objetivo principal é levar informações sobre cidadania, saúde, meio ambiente e qualidade de vida para alunos e comunidades rurais, por meio de atividades pedagógicas e concursos.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável.
